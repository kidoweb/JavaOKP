Каждая лабораторная работа представляет собой Maven проект с стандартной структурой:
- `pom.xml` - конфигурация Maven
- `src/main/java/` - исходный код
- `src/test/java/` - тесты
- `target/` - скомпилированные файлы (игнорируется git)
